2️⃣ Variables et Types

Découvre comment stocker et manipuler des données en PHP.

🌞
Une variable est une boîte qui contient une donnée. En PHP, toutes les variables commencent par le symbole $.

📘 Exemple

<?php
$nom = "Alice";
$age = 25;
$prix = 12.5;
$estAdmin = true;
?>

<p>Nom : <?php echo $nom; ?></p>
<p>Âge : <?php echo $age; ?> ans</p>
<p>Prix : <?php echo $prix; ?> €</p>
<p>Admin ? <?php echo $estAdmin ? "Oui" : "Non"; ?></p>

🧠 Types courants

  • string : texte
  • int : nombre entier
  • float : nombre à virgule
  • bool : vrai/faux
  • array : tableau
  • object : objet

📦 Exemple de tableau

<?php
$fruits = ["pomme", "banane", "fraise"];
echo $fruits[0]; // pomme
?>

🔁 Exemple de boucle sur un tableau

<?php
foreach ($fruits as $fruit) {
    echo $fruit . "<br>";
}
?>
🧩 Les variables seront essentielles pour stocker les données saisies par un utilisateur (formulaire de recherche, etc.).
