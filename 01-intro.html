1️⃣ Introduction à PHP

Découvre ce qu’est PHP et exécute ton premier code dynamique.

🌞
PHP (PHP: Hypertext Preprocessor) est un langage de programmation côté serveur. Il permet de générer du contenu dynamique (HTML, JSON, etc.) avant que la page soit envoyée au navigateur.

📘 Qu’est-ce qu’un langage “côté serveur” ?

Contrairement au JavaScript qui s’exécute dans ton navigateur, PHP s’exécute sur un serveur web (comme Apache ou Nginx). Ce serveur renvoie au navigateur le résultat final, jamais le code source PHP lui-même.

🔧 Exemple simple

<?php
echo "Bonjour Monde !";
?>

Ce code ne s’affichera correctement que si tu le lances dans un environnement serveur (comme XAMPP, WAMP, MAMP ou Docker).

🧠 Explication

  • <?php ouvre une section PHP.
  • echo affiche du texte dans la page.
  • ?> ferme la balise PHP.

⚙️ Tester ton code

Installe un serveur local (par exemple XAMPP) puis place ton fichier dans le dossier htdocs.
Ouvre ensuite ton navigateur et tape : http://localhost/01-intro.html

💡 Exemple un peu plus avancé

PHP peut mélanger du HTML et du code dynamique :

<!DOCTYPE html>
<html>
<body>
<h1>Bonjour</h1>
<p>Aujourd’hui, nous sommes le <?php echo date('d/m/Y'); ?></p>
</body>
</html>

Ici, la fonction date() génère automatiquement la date du jour.

✅ À retenir

  • PHP est interprété par le serveur avant d’être envoyé au navigateur.
  • Tu peux combiner HTML et PHP dans un même fichier.
  • La commande echo sert à afficher du texte ou des variables.
